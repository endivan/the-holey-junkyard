"REFC" Referenced Cells Extension

Fingerprint ID:0x52454643

This fingerprint, from Cat’s Eye Technologies, implements the table of references shared among IPs, which maps scalar value to vector (of same dimension with the space). It provides the following commands:

D : ref — vectorv
Dereferences the scalar value to vector. Reflects if ref is not valid reference.
R : vectorv — ref
Create new reference to vector.
