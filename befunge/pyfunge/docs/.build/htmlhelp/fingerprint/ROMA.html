"ROMA" Funge-98 Roman Numerals

Fingerprint ID:0x524f4d41

This fingerprint, from Cat’s Eye Technologies, implements a few commands pushing huge numbers, which is cumbersome for normal commands. It provides the following commands:

I : — 1
Pushes 1.
V : — 5
Pushes 5.
X : — 10
Pushes 10.
L : — 50
Pushes 50.
C : — 100
Pushes 100.
D : — 500
Pushes 500.
M : — 1000
Pushes 1000.
